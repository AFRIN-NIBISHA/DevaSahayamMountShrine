# DevaSahayamMountShrine

Official website for the St. Devasahayam Martyrdom Place (Our Lady of Sorrows Shrine).

## Project Structure
- **/Client**: React (Vite) frontend application.
- **/Server**: Node.js/Express backend API.

## Features
- Multilingual support (English, Tamil, Malayalam).
- Dark/Light theme toggle.
- Comprehensive Shrine Management section.
- Accommodation information and enquiries.
- Spiritual services booking (Mass, Donations, Prayer Requests).
